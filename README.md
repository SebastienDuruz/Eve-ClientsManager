# Eve-ClientsManager
A simple application build with Python that let me manage the differents **Eve Online** clients opened in a Linux environment.

## Dependancies

Fedora ==>
<code>
  dnf install wmctrl  
</code>

ArchLinux ==>
<code>
  pacman -S wmctrl  
</code>

Ubuntu ==>
<code>
  apt install wmctrl
</code>

## Usage
<code>
  python eveclientsmanager/eveclientsmanager.py
</code>
